Java 21 en pratique

Comprendre ce que les virtual threads changent vraiment.

Ce lab met en scene trois sujets qui reviennent vite en production : le debit des taches I/O, le pinning avec synchronized, et la propagation de contexte.

1

Bench

Comparer un pool de platform threads avec des virtual threads sur une charge I/O simple.

2

Pinning

Visualiser pourquoi un bloc synchronized peut immobiliser le carrier thread en Java 21.

3

Context

Observer les pieges autour des ThreadLocal et des traces dans les workflows concurrents.

En ligne, le site tourne en simulation. En local, tu peux lancer le backend Spring Boot pour obtenir les vraies mesures JVM.
